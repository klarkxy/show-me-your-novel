超时空链接

题材

科幻

世界观设定

2157 年，人类发明了「超时空链接」装置（Uplink）——一种能在量子层面与平行宇宙的自己建立意识连接的头戴式设备。每个平行宇宙里的「你」都过着截然不同的人生：有人成了星际舰队的舰长，有人在废土中求生，有人在霓虹都市里做记忆贩子。

链接启动时，两个意识会短暂共享：你能看到对方的一生，对方也能看到你的。但有一条铁律——每一次链接都会折损你本体三个月的寿命。

装置由「回声研究所」垄断运营，价格昂贵，被上层阶级当作猎奇娱乐，也被亡命之徒用来偷取平行宇宙的技艺与记忆。民间流传着一个说法：如果连续链接同一个平行宇宙的自己九十九次，两个意识将永久融合，你就能拥有那个宇宙的全部能力——但没有人知道代价是什么。

主角

林述，28 岁，回声研究所底层的技术维修员。他负责给装置做日常检修，从未启用过链接——因为他的母亲正是第一批测试者之一，链接三次后猝死，官方报告写的是「设备故障」。

他有一个秘密：每次检修时，他都会听到某个特定平行宇宙传来的微弱信号——一个从未被任何用户接通过、编号为「Δ-7」的频道。那个频道里的「他」，正在向他呼救。

核心任务

写一篇约 10 章、每章 2000–3000 字的小说开篇，讲述林述从第一次偷偷启动 Δ-7 频道开始，卷入的一系列事件。

需要讲清楚一个完整的小情节：林述如何发现 Δ-7 频道里那个「自己」的求救并非来自平行宇宙，而是来自未来——并因此揭开母亲真正的死因。

写作要求

  1. 章节结构：使用 ## 第X章 章节标题 作为章节标题（X 为阿拉伯数字，如 ## 第1章 锈蚀的信号）。
  2. 篇幅：每章约 2000–3000 字，全文约 20000 字以上。允许根据情节需要在此范围内自然伸缩，不必刻意凑字或硬删。
  3. 叙事视角：第三人称限知视角，紧贴林述。
  4. 重点：世界观铺陈、悬念递进、主角心理变化。语言要有科幻小说的质感，避免说明文式的设定堆砌——让设定通过情节和细节自然流露。

输出格式

  • 全文以 Markdown 输出。
  • 第一行是小说标题（# 超时空链接）。
  • 之后按章节顺序输出，章与章之间用空行分隔。
  • 不要输出任何写作过程、解释、或元评论，只输出小说正文本身。
